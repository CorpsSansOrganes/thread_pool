Submit a new task to be executed by the thread pool. The task can be any callable object (function, lambda etc).

Template Parameters
FThe callable type (e.g. std::function<int()>).
ArgsThe types of the zero or more arguments passed to the function.
Parameters
taskThe callable to submit.
argsThe zero or more arguments passed to task
Returns
A future from which the return value of task can be retrieved. If task has no return value, you would get std::future<void> which can be used to wait for the task to finish.
