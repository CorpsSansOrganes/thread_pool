Detailed Description

template<class T, class Container = std::queue<T>>
class EK::WaitableQueue< T, Container >

Waitable, thread-safe queue class.

Template Parameters
Tis the value type which the container holds.
Containeris the container class used. Needs to support the following methods: push(), pop(), front(), empty().
